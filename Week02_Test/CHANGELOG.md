# 프로젝트 변경 이력 (CHANGELOG)

이 파일은 `Week02_Test` 프로젝트의 코드 수정 및 기능 추가 내역을 기록하는 곳입니다.

## [초기 상태] - 2026-04-22
### 현재 구현된 기능
- **캐릭터 생성 (`CreateCharacter`)**: 이름 입력, 직업 선택(전사, 도적, 마법사, 모험가), 하드코어 모드 설정 가능.
- **UI 시스템 (`RenderScene`)**: 3분할 화면 구성 (적 상태, 게임 로그, 캐릭터 상태). `windows.h`를 이용한 콘솔 색상 및 커서 제어.
- **전투 시스템 (`StartBattle`)**:
    - 랜덤 몬스터 생성 (슬라임, 고블린, 오크, 스켈레톤, 울프).
    - 턴제 전투 (일반 공격, 크리티컬 공격).
    - 몬스터 공격력(`atk`) 스탯 반영.
- **성장 시스템 (`LevelUp_Reference`)**: 전투 승리 시 레벨업 기능.
- **아이템 시스템 (`Looting`)**: 전투 승리 후 전리품 획득 (골드, 포션, 검, 갑옷 조각).
- **데이터 구조**: `MonsterData`, `GameData` 구조체를 통한 상태 관리.
