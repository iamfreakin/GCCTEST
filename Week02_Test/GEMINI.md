# Project: Week02_Test (C++ Learning)

이 프로젝트는 C++을 학습하기 위한 용도로 구성되었습니다. Gemini CLI는 다음의 지침을 엄격히 준수해야 합니다.

## 기본 지침
- **언어:** 모든 답변과 설명은 **한국어**로 작성합니다.
- **학습 지원:** 코드 수정이나 기능 구현 시, 해당 코드의 동작 원리와 사용된 C++ 개념(예: 메모리 관리, 객체 지향 원칙 등)을 친절하게 설명합니다.
- **코드 품질:** 학습자가 이해하기 쉽도록 명확하고 표준적인(C++11 이상) 코드를 지향합니다.

## 코딩 스타일 및 규칙
- 변수 및 함수 명명 규칙은 프로젝트의 기존 관례를 따릅니다.
- 복잡한 로직에는 주석을 달아 학습을 돕습니다.
- 가능한 경우 최신 C++ 기능을 활용하는 방법을 제안합니다.
